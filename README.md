# 🍽️ Food Cuisine Recommender System

🚀 **Live Demo:** https://foodcuisinerecommendersystem-jgkkt5kfmtb5hpm8buvqej.streamlit.app/

A machine learning-based recommendation system that suggests similar food dishes based on user selection. The app provides top recommendations along with images using an interactive Streamlit interface.

---

## ✨ Features

* 🔍 Select any food item
* 🤖 Get top 5 similar food recommendations
* 🖼️ Displays food images
* ⚡ Fast & interactive UI
* 📦 Optimized deployment (large files loaded dynamically)

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Streamlit
* Machine Learning (Similarity-based recommendation)

## ⚙️ Installation

```bash

cd Food_Cuisine_Recommender_System
pip install -r requirements.txt
```

---

## ▶️ Run Locally

```bash
streamlit run app.py
```

---

## 📊 How It Works

* Uses a precomputed similarity matrix
* Finds similar food items based on user input
* Displays recommendations with images
* Uses text-based matching for image mapping

---

## ⚠️ Note

* Large file (`similarity.pkl`) is not stored in GitHub
* It is downloaded dynamically at runtime
